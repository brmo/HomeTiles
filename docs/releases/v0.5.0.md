# HomeTiles v0.5.0

This release introduces the new configurable screensaver and refreshes the visual
and editing experience across HomeTiles.

## Highlights

- Dedicated screensaver editor with live preview in the web admin
- JPEG slideshow from `/images` on microSD, with image selection, ordering, shuffle,
  global duration, zoom, and focus controls
- Freely movable and resizable clock with time/date/weekday options, alignment,
  larger fonts, and text shadow
- Sensor, Energy, Switch, Scene, and Media tiles in the bottom two screensaver rows
- Optional tile shadows, subtle borders, and per-tile opacity
- Automatic screensaver timeout in **Settings → Display**; Clock tiles still open it
  immediately
- Hardware-assisted image preparation across the Waveshare 4B, Waveshare 8-inch,
  and M5Stack Tab5 profiles
- New Inter UI font and multiple web-admin usability improvements, including
  multi-file uploads, live entity refresh, persistent tabs, and improved previews
- Screensaver-aware import/export with backward compatibility for older backups
- Aggregated Energy Dashboard values now appear in the web preview
- Stability improvements for PPA rendering, WiFi/SDIO traffic, MQTT commands,
  sensor history, energy statistics, and media covers

## microSD Setup

Create `/images` in the root of a FAT32 microSD card and place `.jpg` or `.jpeg`
files inside. See the full [Screensaver guide](https://galusperes.github.io/HomeTiles/screensaver/).

## Compatibility

The release contains firmware builds for all supported HomeTiles devices. Existing
tile layouts and older JSON exports remain supported.
