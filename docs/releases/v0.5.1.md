# HomeTiles v0.5.1

This maintenance release refines the new screensaver editor and adds a matching
visual option for the normal dashboard.

## Highlights

- Optional subtle tile borders across the complete normal dashboard
- A global **Settings → Display** checkbox that updates every normal web preview
  and every normal on-device tile while keeping the screensaver setting separate
- Fixed the unwanted green focus rectangle shown after clicking web-admin
  checkboxes
- Renamed the screensaver background section from **Screensaver / Wallpapers**
  to **Slideshow / Images** for clearer terminology
- Improved contrast and consistency in the slideshow image controls

## Compatibility

Existing installations default to enabled subtle borders and can turn them off
at any time. Tile layouts, screensaver settings, and older JSON exports remain
compatible. The release contains builds for every supported HomeTiles device.
